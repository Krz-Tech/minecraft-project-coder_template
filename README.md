# Minecraft Development Environment

Krz-Tech Minecraft Server Project 開発環境

## 使い方

```bash
./mc setup    # 初期セットアップ
./mc start    # サーバー起動 + ポートフォワード
./mc stop     # 停止
./mc status   # 状態確認
./mc logs     # ログ表示
```

## 接続

サーバー起動後、Minecraft クライアントで:

```
localhost:25565
```

## Skript 開発

```
minecraft-server/plugins/Skript/scripts/
```

サーバー内: `/skript reload all`
